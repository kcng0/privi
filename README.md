# PrivateHeart Vault

> Personal, offline **Android media vault**. Hides photos & videos from the
> system gallery, rates them **1–3 hearts**, supports favorites, playlist
> playback (built-in or **VLC**), and **PIN + biometric** lock. Dark theme only.
> Sideload-only — no cloud, no accounts, no analytics.

single-owner personal This is a personal project; simplicity is favored over features.

## Status

🚧 **Scaffolding.** Design/architecture docs are complete; app code starts at
[Phase 1](./docs/04-implementation/roadmap.md#phase-1--foundation). The native
`android/` project is generated on first bootstrap (see below).

## Quick start

```bash
# 1. Install Flutter + Android SDK (see DEVELOPMENT.md)
# 2. Generate native scaffold, deps, codegen:
./scripts/bootstrap.sh
# 3. Run on a device:
make run
```

Full setup, troubleshooting, and daily commands: **[DEVELOPMENT.md](./DEVELOPMENT.md)**.

## Tech stack (summary)

Flutter (dark M3) · Riverpod 3 · Drift (SQLite) · video_player + VLC hand-off ·
local_auth + flutter_secure_storage. Min SDK **API 26**. No `INTERNET`
permission. Details: [tech-stack.md](./docs/04-implementation/tech-stack.md).
